**WEATHER FORECASTING APP -** 

Developed an application that provides the weather details such as temperature, humidity, precipitation, etc according to the city mentioned by the user.

Follow the rules below and enjoy the app!

1. Go to files
2. Create a folder and name it as icons and add the photos into it
3. In the top (in downloads only) type cmd (opens the command panel)
4. In that type the following: (To install stuffs)
    python3 -m pip install pyqt5
    python3 -m pip install requests
5. Then type this: (To run app.py)
     python3 app.py
